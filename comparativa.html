Comparativa: HTML vs HTML5

Diferencias clave y evolución del lenguaje web

Características Principales de HTML5

Elementos Multimedia

  • Etiquetas <video> y <audio> para reproducción nativa
  • Elemento <canvas> para gráficos 2D
  • Soporte integrado para gráficos vectoriales SVG
  • Etiqueta <track> para subtítulos en video

Elementos Semánticos

  • <header>, <footer>, <section>
  • <article>, <nav>, <aside>
  • <main> para contenido principal

Formularios Mejorados

  • Nuevos tipos: email, URL, fecha, hora, rango, color
  • Elemento <datalist> para sugerencias
  • Etiqueta <output> para mostrar resultados

Tabla Comparativa: HTML vs HTML5

Característica HTML HTML5
Etiquetas y elementos Etiquetas limitadas sin estructura semántica Nuevas etiquetas semánticas, soporte para formas y JavaScript en segundo plano
Compatibilidad del navegador Compatible con todos los navegadores Compatible con navegadores modernos, limitado en versiones antiguas
Opciones de almacenamiento Cookies temporales y limitadas Caché de aplicación, localStorage, sessionStorage, IndexedDB
Manejo de errores Enfoque inconsistente, difícil depuración Manejo mejorado con semántica estructurada
Soporte móvil Sin soporte móvil integrado Optimizado para dispositivos móviles
Codificación de caracteres Compleja y larga Sencilla y directa
Grafos vectoriales Requiere Flash, Silverlight o VML Soporte nativo con SVG y Canvas
Soporte audio/video Requiere Flash Player Etiquetas nativas <audio> y <video>
Etiquetas obligatorias <html>, <body>, <head> obligatorias Estas etiquetas pueden omitirse

Elementos Modificados o Eliminados

Elementos Modificados

  • <applet> reemplazado por <object>
  • <acronym> cambiado a <abbr>
  • <dir> actualizado a <ul>

Elementos Eliminados

  • <frameset>, <frame>, <noframes>
  • <strike>, <big>, <basefont>
  • <font>, <center>, <tt>

Nota: Los estilos de estos elementos eliminados ahora se manejan con CSS
